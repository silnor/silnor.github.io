# Silnote

**Just notes, no bloat.**<br>Status: *In production*

~~Open~~

Everyone's making a notes app these days, so I thought that I'd join in.

Just kidding. I'm mostly making this for yourself, and you as well if you're interested in it. Obviously, there are some other privacy-focused alternatives like [Notesnook](https://notesnook.com) and [Standard Notes](https://standardnotes.com), which are probably better to use then this, especially since Standard Notes is kinda synced with Proton services too.

I also want this to be a start for your notes app, like how google has chromium to make your own browser. It has nothing, but with your own touch, it can have everything, as least, everything you want.
